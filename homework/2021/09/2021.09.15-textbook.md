## Question 181

\[
f(x) = (x + \cos x)(1 - \sin x)\\
u = (x + \cos x)\\
u' = (1 - \sin x)\\
v = (1 - \sin x)\\
v' = - \cos x\\
f'(x) = (1 - \sin x)(1 - \sin x) + - \cos{x}(x + \cos x)\\
f'(x) = (1 - \sin x)^2 - x \cos x - \cos^2{x}
\]

## Question 197

\[
f(x) = -3(\sin x)(\cos x)\\
f'(x) = 3(\cos x)(\cos x) + (-\sin x)(\sin x)\\
f'(x) = 3(\cos^2 {x} - \sin^2{x})\\
3(\cos^2 {x} - \sin^2{x}) = 0\\
\cos^2 {x} - \sin^2{x} = 0\\
1 - \sin^2 {x} - \sin^2{x}\\
1 - 2\sin^2{x} = 0\\
-2\sin^2{x} = -1\\
\sin^2{x} = \frac{1}{2}\\
\sin{x} = \sqrt{\frac{1}{2}}\\
\sin{x} = +/- \frac{\sqrt2}{2}\\
\frac{\pi}{4}, \frac{3\pi}{4}, \frac{5\pi}{4}, \frac{7\pi}{4}
\]

## Question 198

- \(f(x) = x - 2\cos{x}\)
- \(f'(x) = 1 + 2 \sin x\)
  - \(2 = 1 + 2\sin{x}\)
  - \(1 = 2\sin{x}\)
  - \(\sin{x} = \frac{1}{2}\)
  - \(\sin^{-1}{\frac{1}{2}} = \frac{\pi}{6}\)
  - \(\pi - \frac{\pi}{6} = \frac{5\pi}{6}\)
- \(\frac{\pi}{6}, \frac{5\pi}{6}\)
